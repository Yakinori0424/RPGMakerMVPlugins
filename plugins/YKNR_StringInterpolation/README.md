# YKNR_StringInterpolation.js [Ver.1.0.0]
String クラスに、式展開した文字列を返す関数を追加します。  
  
バッククオート(``)を用いても式展開可能ですが、  
ツクールMV内の文字列は('' or "")がメインなため、  
文字列からそれっぽく展開できるようにしました。
